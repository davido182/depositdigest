# 🚀 RentaFlux - Instrucciones de Distribución 
 
## 📱 PARA ANDROID (APK): 
 
1. Localiza el APK generado: 
   📍 android\app\build\outputs\apk\debug\app-debug.apk 
 
2. Sube el APK a: 
   • Google Drive (recomendado) 
   • Dropbox 
   • Tu servidor web 
 
3. Obtener enlace publico de descarga 
 
4. Compartir con amigos usando el mensaje preparado 
 
## 🌐 PARA IPHONE + ANDROID (PWA): 
 
OPCION 1 - Servidor local (para pruebas): 
1. Ejecuta: npm run dev 
2. Obtén tu IP local: ipconfig 
3. Comparte: http://tu-ip:8081 
 
OPCION 2 - Cloudflare Tunnel (global): 
1. Configura token en .env.local 
2. Ejecuta: start-with-cloudflare.bat 
3. Tu app estará en: https://rentaflux.tu-dominio.com
